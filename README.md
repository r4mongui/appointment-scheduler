Appointment Scheduler

Sistema de Agendamento — Trabalho de Arquitetura e Padrões de Projeto

Este projeto implementa um sistema simples de agendamento, utilizando
Node.js + Express + TypeScript.

TECNOLOGIAS: - Node.js - Express - TypeScript - SQL.js - ts-node-dev

INSTALAÇÃO: 
1. git clone https://github.com/r4mongui/appointment-scheduler.git
2. cd appointment-scheduler 
3. npm install 
4. npm run dev

ENDPOINTS: GET /appointments POST /appointments DELETE /appointments/:id
